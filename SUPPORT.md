# Support Guide

If you need help or have questions about Turnitout, please use the following resources:

## 🔍 Frequently Asked Questions
Before reaching out, check the FAQ section in [GETTING_STARTED.md](GETTING_STARTED.md) or look at existing issues.

## 🐛 Bug Reports & Feature Requests
If you found a bug or have a suggestion:
1. Check the [GitHub Issues](https://github.com/AhmadHassan-BTed/Turnitout/issues) to see if it has already been reported.
2. If not, open a new issue using our templates.

## 💬 Discussions & Q&A
For general questions, configuration help, and discussions:
- Visit our [GitHub Discussions](https://github.com/AhmadHassan-BTed/Turnitout/discussions) tab.
